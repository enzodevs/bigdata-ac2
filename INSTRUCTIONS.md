# Instrucoes do Projeto Final - Big Data

## Introducao

O projeto final do curso de Big Data tem como objetivo permitir que os alunos apliquem os conhecimentos adquiridos durante o semestre em um projeto pratico e desafiador. Neste projeto, voces trabalharao em equipes de quatro pessoas para desenvolver sistemas de analise de dados com tecnologias de Big Data que envolvem analise exploratoria de dados (EDA), pre-processamento e modelos preditivos.

## Descricao do Projeto

Voces tem a liberdade de escolher uma base de dados de sua preferencia no Kaggle, na qual o conjunto de dados destinado a tarefa pode ser do tipo **regressao** ou **classificacao**, a escolha ficara por conta da equipe. O projeto deve ser dividido em tres partes principais:

---

### Parte 1: Construcao de um Ambiente de Big Data (1 ponto)

Construir um ambiente de big data com ecossistema Apache, utilizando o Docker para simular um sistema de cluster de big data.

### Parte 2: Tarefa de Analise de Dados (5 pontos)

Selecionar uma base de dados adequada para uma tarefa de regressao ou classificacao.

Exemplos: previsao de precos de imoveis, previsao de vendas, classificacao de produtos, avaliacao de sentimentos em revisoes, etc.

**Pre-requisito: o dataset precisa ter mais de 1GB de dados.**

Etapas:

1. **Selecao de Dados** - Escolha uma base de dados apropriada no Kaggle. A base precisa passar por aprovacao do professor.
2. **Pre-processamento** - Tratamento de desequilibrio de classes e outras etapas relevantes.
3. **Analise Exploratoria de Dados (EDA)** - Extrair insights e entender a distribuicao das classes.

### Parte 3: Tarefa de Previsao (4 pontos)

Treinar os seguintes algoritmos de aprendizado de maquina:

- **Decision Tree**
- **Linear Regression** (ou Logistic Regression)
- **Redes Neurais**

Etapas:

1. **Modelos de Regressao/Classificacao** - Implementar os 3 modelos de predicao para a tarefa selecionada na Parte 2.
2. **Avaliacao e Apresentacao** - Avaliar o desempenho dos modelos e preparar uma apresentacao detalhando processo, resultados e conclusoes.

---

## Apresentacao Final

**Datas:** 11/05, 12/05, 25/05 e 26/05

Cada equipe apresentara seus resultados e conclusoes. A apresentacao deve incluir:

- Demonstracoes praticas dos modelos
- Analises de resultados
- Desafios enfrentados
- Sugestoes para melhorias futuras

A avaliacao considera tanto o desempenho dos modelos quanto a qualidade da apresentacao.
